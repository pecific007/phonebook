# Phonebook:

Phonebook is a demo for my python library for sqlite3: [plibsqlite](https://github.com/pecific007/plibsqlite). It is meant to show the following capabilities of the library:

- Creating table
- Inserting values into table
- Selecting from the table
- Updating values
- Deleting from the table

### Run:
> ```sh
> $ flask run
> ```
to run the flask server. It uses the default port `:5000`
